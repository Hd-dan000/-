# 实训评价系统 · 设计说明

## 作品简介

面向软件工程实训场景的AI智能评价系统。跳出传统"上传—打分"线性流程，围绕教师减负与学生赋能两大痛点，构建**多智能体协作评价架构**——代码规范、逻辑漏洞、文档质量等专项评审Agent并行工作，汇总智能体统合输出，实现媲美资深工程师的深度评阅。同时首创**校企双轨评价体系**，一份成果同步输出教学标准与企业岗位能力两份报告，将评价结果映射到企业能力图谱，精准标注达标情况与短板。系统植入"提交—代码审查—评价—复盘"的企业级开发流程，还原真实职场场景。

## 作品亮点·创意·文化·AI工具（200字内）

**亮点**：多Agent协作深度评阅 + 校企双轨评价 + 共性问题聚类辅助教学。**创意**：AI从打分工具升维为教学伙伴——智能评语替代分数、异常预警、个性化学习路径推荐，让评价成为学习环节。**文化**：以评促学，融合工匠精神，实训即对接职场标准；成长轨迹雷达图传递"看见进步、尊重差异"的人文理念。**AI工具**：微调开源大模型+本地知识库嵌入校方标准，Ant Design组件体系支撑前端交互。

## 技术架构

- **多智能体协作**：代码规范、逻辑漏洞、文档质量等专项Agent + 汇总Agent
- **大模型微调**：基于开源模型，用软件工程领域代码数据进行领域适配
- **本地知识库**：嵌入校方评分标准、企业代码规范
- **工程化实践**：完整需求/设计/测试文档，一键部署，高并发保障
